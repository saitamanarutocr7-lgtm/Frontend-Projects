Marketplace Basic UI Dashboard

📝 Overview
This is a study of a modern E-commerce interface focusing on Sport & Equipment categories. This project marks my transition from basic linear layouts to complex, modular dashboard designs using advanced CSS techniques.

🛠️ Skills Demonstrated
Complex Grid Systems: Managed a multi-column product layout that maintains its shape even with dynamic content.

UI Modularity: Designed a sidebar where every filter (Brand, Price, Rating) is an independent "card" component.

Advanced Positioning: Successfully placed "Top Item" badges over images using absolute positioning.

Checkbox Alignment: Solved the layout challenge of aligning labels and checkboxes on opposite sides of a container.

🧪 Challenges Faced
Image Distortion: Fixed issues where product images were stretching by implementing object-fit: contain.

Spacing Logic: Switched from messy margins to the gap property for consistent internal spacing between modules.
